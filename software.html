C++ OpenGL Graphics

OpenGL Ray Tracer

OpenGL Ray Tracer (2019) is a graphics program that was created for CSE 287: Introduction to Computer Graphics and Games. For this project, our goal was to use the open-source C++ graphics library OpenGL to create a basic ray tracer for rendering 3D scenes. Some of the features in this ray-tracer include ambient, directional, positional, and spot lights, shadows created using "shadow feelers", inter-object reflections between objects, multiple camera angles, and anti-aliasing.

Vertex-Based Graphics Engine

Vertex-Based Graphics Engine(2019) is a graphics program that was created for CSE 287: Introduction to Computer Graphics and Games. This project was meant to show off vertex-based graphics pipeline, which is used in most 3D game engines. Some of the features of this project include basic animaiton, per vertex / per fragment shading, perspective / orthographic projections, a free-moving camera, object clipping, and backface culling.

Java

Command-Line Tic-Tac-Toe

Command-Line Tic-Tac-Toe (2018) is a project that was done for CSE 274: Data Structures and Abstractions. This is a game of Tic-Tac-Toe with a pseudo-AI that works by using a dictionary data structure. First, a list of all possible Tic-Tac-Toe boards are generated. Then, the boards are stored in a dictionary ADT, using the board configuration as the key and the next best move for that board as the value.
