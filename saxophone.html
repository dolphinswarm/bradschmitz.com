Classical Performances

Galaxy Wars - Main Theme

Galaxy Wars - Main Theme (2017) is, as the title implies, the main theme for a scrapped video game I was working on called Galaxy Wars. When working on video games, I have a nasty habit of wanting to write the music before completing the framework of the game. This piece is a prime example of that: it was meant to play on the title screen and a few other areas of the game, but it was written before any place it could be used was created. Maybe one day I'll keep working on the game and be able to use this theme?

Pixel Jet - Battle Theme

Pixel Jet - Battle Theme (2013) is the theme for the Arcade Mode mode (the standard mode) in Pixel Jet. Interestingly enough, this song was not written in FamiTracker like Pixel Jet - Main Theme, and was instead created by composing a MIDI file and using a program called GXSCC to give each instrument a chiptune soundfont. Admittedly, the chiptune sound of GXCSS is less authentic, but it is much easier to use than FamiTracker when composing new songs.
